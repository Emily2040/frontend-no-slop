# Task
Audit an internal analytics dashboard used by support leads.

# Current layout
- Top row has four KPI cards.
- Filters are hidden behind a funnel icon.
- Main content is a chart followed by a ticket table.
- Row actions only appear on hover.
- Empty states are blank white panels.

# Known complaints
- New users cannot tell which date range is active.
- Keyboard users miss row actions.
- The page feels busy even though some controls are hidden.

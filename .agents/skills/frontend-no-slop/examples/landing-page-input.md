# Task
Refactor a B2B landing page for QueryDock, a warehouse query cost monitor.

# Audience
Data platform leads and engineering managers.

# Problems
- The current page says "observability for the modern data stack" and never explains what the product actually does.
- The hero uses a giant abstract gradient with no product evidence.
- There are three CTAs in the first viewport.
- Feature cards are vague and the page buries pricing and proof.

# Constraints
- Tone must stay plain and technical.
- Primary CTA: "Book demo"
- Secondary CTA: "See query sample"
- Avoid glassmorphism and decorative neon.
- Keep the first viewport concise on 1440px and readable on mobile.
- Use existing design tokens: 8px spacing scale, 12px radius, neutral palette with one accent color.
